> **StudyToTech** é uma plataforma web inovadora desenvolvida para democratizar o acesso a ferramentas de organização acadêmica, ajudando estudantes a gerenciarem seu tempo e melhorarem seu desempenho através de tecnologia acessível e inteligente.

---

## 📌 Visão Geral do Projeto

Este projeto faz parte do **Projeto Integrado Multidisciplinar (PIM)** da UNIP (2026). O StudyToTech surge para resolver a dificuldade de gestão de tempo enfrentada por muitos estudantes e a falta de ferramentas gratuitas de alta qualidade no mercado brasileiro.

### ✨ Funcionalidades Principais
- 📅 **Planejador Inteligente:** Organize disciplinas, provas e entregas em um só lugar.
- 🧠 **Assistente de IA:** Recomendações personalizadas de estudo com base no histórico do usuário.
- ⏱️ **Modo Pomodoro:** Timer integrado para otimizar o foco e a produtividade.
- 📊 **Dashboard Analítico:** Visualize seu progresso através de gráficos de desempenho.
- 🎮 **Gamificação:** Sistema de conquistas e níveis para incentivar a constância.

---

## 🛠️ Tecnologias e Metodologia

O desenvolvimento seguiu padrões rigorosos de engenharia de software:

- **Metodologia Ágil:** Uso de **Scrum** com ciclos de Sprints quinzenais.
- **Arquitetura:** Foco em escalabilidade e segurança (em conformidade com a LGPD).
- **Interface:** Design centrado no usuário (UX/UI), responsivo e acessível (suporte a LIBRAS).

### 📋 Requisitos do Sistema (Exemplos)
| ID | Descrição | Tipo |
| :--- | :--- | :--- |
| **RF06** | Cadastro de tarefas e avaliações | Funcional |
| **RF22** | Recomendação automática de conteúdos via IA | Funcional |
| **RNF04** | Tempo de resposta inferior a 2 segundos | Não Funcional |
| **RNF06** | Criptografia de dados sensíveis do usuário | Não Funcional |
